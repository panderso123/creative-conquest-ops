# Creative Conquest — Ops System

The operating backbone for the Creative Conquest short-form video agency: the system that lets a VA run delivery while the founder stays on sales. Built from *Creative Conquest — The Playbook*.

## Contents

| File | What it is |
|------|------------|
| `Client_Onboarding_Checklist.docx` | Day-1 intake → first post. Logins, brand kit, strategy, content day, systems, kickoff, Week 1. |
| `Content_Production_Tracker.xlsx` | 20-videos/month pipeline (film → edit → QA → approve → schedule), live Pipeline Summary, weekly posting calendar. |
| `Monthly_ROI_Report.xlsx` | Client-facing scorecard. Fill blue cells; engagement rate, attributed revenue, ROI multiple, cost-per-consult auto-calculate. |
| `Core_Operating_SOPs.docx` | The four delegatable procedures: editing/captions, scheduling, engagement/DMs, prospecting/outreach. |

## How to use

- Duplicate the ROI report per client, per month.
- Open the SOP doc in Word and run **Update Field** on the table of contents to populate it.
- ROI report defaults avg. client value to $2,250 (playbook range $1,500–$3,000); adjust per client at onboarding.
